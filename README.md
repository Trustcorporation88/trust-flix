# JETBOT V7 — Robô Vendedor Automático

Plataforma Node.js para Square Cloud com setup web, catálogo explicativo, WhatsApp local ou Evolution API, pagamentos, entrega, CRM, suporte, automações, funil de presentes, campanhas, leads e integrações.

Comece pelo arquivo `00_COMECE_AQUI_JETBOT_V7.txt`.

## Comandos

```bash
npm ci
npm run check
npm test
npm start
```

## Acesso

O login é definido por `PANEL_USERNAME` e `PANEL_PASSWORD` nas variáveis de ambiente.

## Segurança

Não publique `.env`, tokens, `data/.app-secret`, `.wwebjs_auth` ou `.wwebjs_cache`.
